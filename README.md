# 📈 CorpusQ - Smart Retirement Planner

### 🚀 From Investment to Retirement

A modern retirement planning web application that helps users estimate
their future financial needs using retirement projections, Monte Carlo
simulations, and asset allocation strategies.

------------------------------------------------------------------------

## 📖 Overview

**CorpusQ** is a financial planning platform designed to simplify
retirement planning.

Users can enter their income, investments, retirement goals, and
lifestyle expectations to receive a detailed retirement projection while
considering inflation and investment growth.

------------------------------------------------------------------------

-   👤 Personal Information Collection
-   💼 Employment Status Selection
